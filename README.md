# Recruitment Performance Dashboard | Power BI

## Project Overview

This project is an HR and recruitment analytics dashboard developed using Power BI. The dashboard analyses employee performance, recruitment effectiveness, hiring costs, education background, and job satisfaction to help organisations make data-driven HR decisions.

The dashboard provides interactive visualisations that allow users to identify trends in employee performance and evaluate recruitment strategies.

---

## Dashboard Features

### Recruitment Analysis

* Recruitment Cost by Performance
* Cost of Hire by Position
* Method of Selection by Performance

### Employee Performance Analysis

* Position by Performance
* Gender by Performance
* Employee Count per Position

### HR Insights

* Job Satisfaction by Performance
* Job Satisfaction by Position
* Education by Position
* Training Undertaken by Position

---

## Key Business Questions Answered

* Which positions have the highest recruitment cost?
* Which recruitment methods produce better-performing employees?
* How does job satisfaction vary across positions?
* What is the relationship between education and employee roles?
* Which employee groups demonstrate stronger performance metrics?
